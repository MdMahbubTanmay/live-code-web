## 📌 Project Background

This is my **first React TypeScript project**.

I built this project mainly for **personal learning and exploration**.  
I am not primarily focused on web development  this repository is part of my effort to learn modern frontend tools by building a real, practical application.

Before this, I created a desktop version in JavaFX. After that experience, I wanted to implement the same core concept for the web to understand:

- component-based UI with React
- strict typing with TypeScript
- modern build tooling with Vite
- browser-first software delivery (no install required)

## 🚀 Getting Started

### Prerequisites

- **Node.js** (LTS recommended)
- **npm**

### 1) Clone the repository

```bash
git clone https://github.com/MdMahbubTanmay/live-code-web.git
cd live-code-web
```

### 2) Install dependencies

```bash
npm install
```

### 3) Start development server

```bash
npm run dev
```

Open the URL shown in your terminal (typically `http://localhost:5173`).

### 4) Build for production

```bash
npm run build
```

### 5) Preview production build

```bash
npm run preview
```
